# deku-packery

> Packery component for [Deku](https://github.com/dekujs/deku)


## Install

```
$ npm install --save deku-packery
```


## Usage

```js
import Packery from 'deku-packery';

export function render() {
	return (
		<Packery>
			<img src='cat.jpg' alt=''/>
			<img src='unicorn.jpg' alt=''/>
		</Packery>
	);
}
```
